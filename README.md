# nw-normalize-menu

Normalize menu(on mac) for nw.js(node-webkit).

## Installation

    npm install nw-normalize-menu

## Usage

load script from html.

```html
<script src="node_modules/nw-normalize-menu/nw-normalize-menu.js"></script>
```
